# % Listas /% /#

## Lista padrão /#
Lista:

[ 
	- elemento 1
	- elemento 2
	- elemento 3
	- $ elemento a negrito na lista /$
/]

	- elemento fora do contexto de lista

## Lista numerada /#

Lista:

[N 
	- elemento 1
	- elemento 2
	- elemento 3
	- $ elemento a negrito na lista /$
/N]
